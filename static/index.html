What Is Brainfuck?

Brainfuck is an esoteric language invented in 1993 by Urban Muller. It uses only eight symbols to manipulate a memory tape and print output. Even with this tiny syntax, it is Turing complete.

Any character that is not one of the eight commands is ignored, so comments and spacing are allowed. This makes it a fun language for puzzles and code golf.

Brainfuck Command Reference

>

Move pointer one cell to the right.

<

Move pointer one cell to the left.

+

Increment current cell value by 1.

-

Decrement current cell value by 1.

.

Output current cell as an ASCII character.

,

Read one input byte into current cell.

[

If cell is zero, jump forward to matching ].

]

If cell is non-zero, jump back to matching [.
